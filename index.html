I LOVE DOING PROJECTS TOO

MedHist

  • Tech Used: NodeJS, JavaScript,MongoDB, HTML5, CSS3, Git
  • MedHist keeps track of your entire medical history so that you don't pay for unnecessary medical costs.
  • It registers certified hospitals all over the world and all these hospitals coordinate with each other and accept each other's reports.
  • This way, if you're travelling and face some unforseen medical circumstances, certified hospitals already know your medical history through MedHist and hence lay focus on treatment rather than repeating medical tests.

Caffeine

  • Tech Used: Muse Headband, Java, Android Studio, Git
  • Winning project at Start Hacks '17
  • Caffeine uses the Muse Headband to gather brain-wave and accelerometer data and then smartly analyses it in an Android app.
  • It triggers an alarm and a strong vibration when a driver dozes off while driving.
  • Caffeine can help prevent more than 100,000 accidents in Canada every year.
  • Future: Planning to integrate Caffeine in the IoT project where it coordinates with the car system to prevent starting the car, hence preventing further accidents.
